# Changelog

All notable changes to the BanglaCode VS Code extension will be documented in this file.

## [1.0.0] - 2026-02-07

### Added
- Initial release of BanglaCode VS Code extension
- Full syntax highlighting for BanglaCode language
  - Keywords highlighting (dhoro, jodi, kaj, etc.)
  - Built-in functions highlighting
  - String and number literals
  - Comments
  - Operators
- IntelliSense and code completion
  - All 25+ keywords with descriptions
  - All 40+ built-in functions with Bengali meanings
  - Auto-detection of user-defined variables
  - Auto-detection of user-defined functions
  - Auto-detection of user-defined classes
- Code snippets (35+ snippets)
  - Control flow: jodi, nahole, ghuriye, jotokkhon
  - Functions: kaj, ferao
  - Classes: class, notun
  - Error handling: chesta, dhoro_bhul, felo
  - Modules: ano, pathao
  - HTTP: server, anun
  - File I/O: poro, lekho
  - And more...
- Hover documentation
  - Bengali meanings for all keywords
  - Descriptions for all built-in functions
- Custom file icon for .bang files
- Language configuration
  - Auto-closing brackets and quotes
  - Comment toggling with //
  - Proper indentation rules

### Technical Details
- Minimum VS Code version: 1.74.0
- Extension size: ~50KB
- No external dependencies
