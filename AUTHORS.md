## Acknowledgments

### Inspiration

BanglaCode was inspired by:

- **Writing An Interpreter In Go** by Thorsten Ball — The book that taught the fundamentals of interpreter design
- **Crafting Interpreters** by Robert Nystrom — Additional techniques and patterns
- **The Go Programming Language** — For providing an excellent implementation language
- **Bengali-speaking students everywhere** — The motivation behind the project

### Special Thanks

- The open source community for invaluable tools and libraries
- Early testers and feedback providers
- Everyone who has starred, forked, or shared the project

## How to Become a Contributor

We welcome contributions from everyone! Here's how to get started:

### Code Contributions

1. **Find an issue** — Look for issues labeled `good first issue` or `help wanted`
2. **Fork the repository** — Create your own copy to work on
3. **Make your changes** — Follow the guidelines in [CONTRIBUTING.md](CONTRIBUTING.md)
4. **Submit a pull request** — We'll review and merge your contribution

### Documentation

1. **Improve existing docs** — Fix typos, clarify explanations, add examples
2. **Write tutorials** — Create guides for specific features or use cases
3. **Translate** — Help translate documentation to Bengali

### Other Ways to Contribute

- **Report bugs** — Help us find and fix issues
- **Suggest features** — Share your ideas for improvements
- **Answer questions** — Help others in GitHub Discussions
- **Share the project** — Spread the word about BanglaCode

## Translations

We're looking for help translating BanglaCode documentation into Bengali (বাংলা).

### How to Help

1. Check the [translation tracking issue](https://github.com/nexoral/BanglaCode/issues) (if exists)
2. Pick a document to translate
3. Submit a PR with the translated document

### Translation Guidelines

- Maintain technical accuracy
- Use commonly understood Bengali programming terms
- Keep code examples in Banglish (as per language design)
- Note: Comments can be translated to Bengali script
